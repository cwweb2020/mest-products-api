# TESLO API

1. clonar proyecto

2. npm install

3. clonar el archivo env tamplate renombrarlo a env

4. cambiar las variables

5. levantar la base de datos

```
docker-compose up -d
```

6. levantar modo desarrollo

```
  npm run start:dev
```
